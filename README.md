# Language Flashcards

Готовая статическая версия для GitHub Pages.

## Публикация
1. Загрузите все файлы этого архива в корень репозитория `flashcards-app`
2. В GitHub откройте Settings → Pages
3. Source: Deploy from a branch
4. Branch: main / (root)

## Что обновлено
- озвучка читает текущую видимую сторону карточки
- в режиме ввода можно отдельно озвучивать слово и перевод
- в базе слов есть отдельные кнопки озвучки для обеих сторон
